# Public Outputs

This folder contains selected public summary outputs generated from the strict EIA-860M battery classifier.

These files are source-boundary constrained outputs, not final market-total estimates.

## Files

### active_operating_by_state.csv

State-level summary of strict battery records with operating status.

### planned_pipeline_by_state.csv

State-level summary of strict battery records with planned status.

### strict_all_statuses_by_status.csv

Status-level summary of all strict battery classifier records.

## Interpretation boundary

Any numeric statement based on these files should use this boundary:

> Based on a strict EIA-860M classifier output, not a final market-total estimate.

These files should not be used as standalone evidence for forecasts, investment conclusions, policy conclusions, grid-stress conclusions, or definitive national deployment totals.
